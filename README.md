# 🦊 Грибо.Net - Браузер с AI Лисичка

Современный веб-браузер с интегрированным AI ассистентом **Лисичка.AI** для чата и голосовых звонков.

## ✨ Основные возможности

### 🌐 Встроенный браузер
- **Адресная строка** с автоматическим добавлением протокола
- **История навигации** (назад, вперед, обновить)
- **Загрузка любых веб-сайтов** через iframe
- **Удобная навигация** с кнопками управления

### 💬 Чат с Лисичка.AI
- **Текстовое общение** с нейросетью
- **Сохранение истории** разговоров в localStorage
- **Быстрая отправка** сообщений (Enter или кнопка)
- **Автоматическая прокрутка** к последнему сообщению
- **Поддержка многострочного ввода** (Shift+Enter)

### 📞 Голосовые звонки
- **Начало звонка** одним кликом
- **Доступ к микрофону** браузера
- **Визуализация звука** в реальном времени (аудиограф)
- **Транскрибирование** разговора
- **Статус звонка** в реальном времени

### ⚙️ Гибкие настройки
- **API ключ Лисичка.AI** - введите ваш уникальный ключ
- **Пользовательский API URL** - измените адрес сервера
- **Сохранение в localStorage** - настройки сохраняются автоматически

## 🚀 Быстрый старт

### 1. Клонирование репозитория
```bash
git clone https://github.com/ListStudio/Gribo.net.git
cd Gribo.net
```

### 2. Открытие приложения
- Просто откройте файл `index.html` в веб-браузере
- Или используйте локальный сервер:
```bash
python -m http.server 8000
# или
npx http-server
```

### 3. Настройка Лисичка.AI
1. Откройте вкладку **⚙️ Настройки**
2. Введите ваш **API ключ** от Лисичка.AI
3. (Опционально) Измените **API URL** если нужно
4. Нажмите **Сохранить**

### 4. Использование

#### Браузер
- Введите URL в адресную строку (например: `google.com`)
- Нажмите **Перейти** или **Enter**
- Используйте кнопки навигации: ←, →, ⟲

#### Чат
- Напишите сообщение в поле ввода
- Нажмите **Отправить** или **Enter**
- Лисичка ответит автоматически

#### Звонки
- Нажмите **Начать звонок**
- Разрешите доступ к микрофону
- Разговаривайте с Лисичкой
- Нажмите **Завершить звонок** когда закончите

## 🔌 API интеграция

### Чат API
```javascript
POST /chat
Headers:
  Authorization: Bearer YOUR_API_KEY
  Content-Type: application/json

Body:
{
  "message": "Привет, Лисичка!",
  "conversation_id": "conv_12345"
}

Response:
{
  "response": "Привет! Как дела?"
}
```

### Звонок API
```javascript
POST /call
Headers:
  Authorization: Bearer YOUR_API_KEY

Body:
{
  "action": "start",
  "audio_stream": WebRTCStream
}
```

## 💾 Локальное хранилище

Приложение использует `localStorage` для сохранения:
- `lisichka_api_key` - API ключ
- `lisichka_api_url` - URL сервера
- `chat_history` - История чатов (JSON)
- `conversation_id` - ID текущего разговора

## 🎨 Дизайн

- **Цветовая схема**: Фиолетовый градиент (#667eea → #764ba2)
- **Макет**: CSS Grid (браузер + панель управления)
- **Адаптивность**: Responsive на мобильных устройствах
- **Анимации**: Плавные переходы и появление сообщений

## 🔐 Безопасность

⚠️ **Важно**: Никогда не делайте ваш API ключ публичным!

- Ключ хранится локально в браузере
- Не передается никому, кроме API сервера
- При необходимости смените ключ в настройках

## 🐛 Известные ограничения

1. **iframe** - может не загружать некоторые сайты с CORS политикой
2. **Звонки** - требуют поддержки Web Audio API
3. **Микрофон** - требует разрешения пользователя (HTTPS в production)

## 📱 Мобильная оптимизация

На мобильных устройствах:
- Браузер и панель управления располагаются вертикально
- Сенсорное управление кнопками и полями ввода
- Автоматическое сокращение высоты компонентов

## 🤝 Разработка

### Структура кода

**HTML** - Разметка UI с элементами:
- `browser-section` - основной браузер
- `chat-section` - чат компонент
- `call-section` - звонки компонент
- `settings-section` - настройки

**CSS** - Стили с использованием:
- CSS Grid для макета
- Flexbox для компонентов
- CSS переменные для цветов
- Media queries для адаптивности

**JavaScript** - OOP с классами:
- `GriboBrowser` - управление браузером и историей
- `LisichkaChat` - работа с чатом и API
- `LisichkaCall` - управление звонками и Web Audio API
- `Settings` - сохранение конфигурации
