⚙️ Конфигурация Грибо.Net

1. Получение API ключа Лисичка.AI

  1. Перейдите на Лисичка.AI
  2. Зарегистрируйтесь или войдите в аккаунт
  3. Перейдите в раздел "API ключи"
  4. Создайте новый API ключ
  5. Скопируйте ключ (выглядит примерно так: sk_live_xxxxx)

2. Добавление ключа в приложение

Есть два способа:

Способ 1: Через интерфейс приложения

  1. Откройте index.html в браузере
  2. Кликните на кнопку ⚙️ Настройки
  3. В поле "API ключ Лисичка.AI" вставьте ваш ключ
  4. Нажмите Сохранить

Способ 2: Программно (в коде)

Добавьте в app.js после строки конфигурации:

// Автоматическая установка ключа
localStorage.setItem('lisichka_api_key', 'sk_live_ваш_ключ_здесь');
localStorage.setItem('lisichka_api_url', 'https://api.lisichka.ai');

3. Проверка конфигурации

  1. Откройте консоль браузера (F12 → Console)
  2. Выполните команду: 
    console.log(localStorage.getItem('lisichka_api_key'))
  3. Если ключ отображается - конфигурация успешна ✅

4. Переменные конфигурации

CONFIG = {
    apiKey: 'sk_live_xxxxx',      // API ключ Лисичка.AI
    apiUrl: 'https://api.lisichka.ai'  // URL сервера
}

5. LocalStorage переменные

Переменная Назначение Пример
lisichka_api_key API ключ sk_live_xxxxx
lisichka_api_url URL API сервера https://api.lisichka.ai
chat_history История чатов (JSON) [{"sender":"user","text":"Привет"}]
conversation_id ID разговора conv_1234567890

6. Примеры API запросов

Чат

fetch('https://api.lisichka.ai/chat', {
    method: 'POST',
    headers: {
        'Authorization': 'Bearer sk_live_xxxxx',
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        message: 'Привет!',
        conversation_id: 'conv_123'
    })
})
.then(r => r.json())
.then(data => console.log(data.response))

Звонок

fetch('https://api.lisichka.ai/call/start', {
    method: 'POST',
    headers: {
        'Authorization': 'Bearer sk_live_xxxxx',
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        action: 'start',
        audio_sample_rate: 16000
    })
})

7. Проверка подключения

// Тест подключения к API
async function testConnection() {
    try {
        const response = await fetch(
            `${CONFIG.apiUrl}/health`,
            {
                headers: {
                    'Authorization': `Bearer ${CONFIG.apiKey}`
                }
            }
        );
        if (response.ok) {
            console.log('✅ Подключение успешно!');
        } else {
            console.log('❌ Ошибка подключения');
        }
    } catch (error) {
        console.error('Ошибка:', error);
    }
}

// Вызовите в консоли:
testConnection();

8. Troubleshooting

Проблема Решение
❌ API ключ не работает Проверьте, что ключ скопирован полностью без пробелов
❌ Чат не отправляет сообщения Убедитесь, что API ключ установлен в настройках
❌ Звонок не работает Разрешите доступ к микрофону браузером
❌ Ошибка CORS Проверьте что URL API содержит https://

9. Безопасность

⚠️ Важно! Никогда не делитесь вашим API ключом! Храните его в безопасности и удаляйте из истории коммитов.
